# ThriveApp
A personal operating system that shows how your daily decisions shape your future.
ThriveApp is a personal operating system that helps users manage their finances, health, and goals — all in one place.

## 🚀 Vision
Instead of using multiple apps, ThriveApp connects your daily decisions to your long-term outcomes.

## 🔥 What it will do
- Track daily habits
- Track expenses
- Track personal goals
- Show how your actions impact your future

## 🧠 Future features
- AI insights on behavior
- Goal projections
- Smart recommendations
